Operations - Drilling

Drilling involves the steps described below. If drilling operation follows insulation milling operations, some steps might be omitted.

Opening files and selecting orientation

Open files and select board orientation following advices of Preparation – File selection and Preparation – PCB orientation sections.

Machine homing

Home your machine as explained in Preparation – Homing section.

Selecting operations

Placing PCB laminate on the machine bed

As instructed, put PCB laminate and spacer on the machine bed and select which size of laminate you use. You can use Move head away button to get machine head out of the way.

Inserting drill

As instructed by the application, insert the drill of specified diameter in your machine’s spindle.

Establishing X and Y offsets

X and Y offsets determine the location of lower left corner of your board on the laminate.

Show me where it is button moves the machine head to the specified location. Should you wish to be very precise in pinpointing the location on the laminate, Move Z axis pane allows you to lower the head, moving the tool closer to laminate surface.

Watch out – you can crash the drill into laminate if you command it to go too low!

Drilling

The screen shows PCB outline and all the holes matching the diameter of the drill currently installed.

You can select, which holes are going to be drilled. Holes can be selected with mouse. Selected holes can be enabled and disabled with toolbar buttons or with keyboard shortcuts: Ctrl+E (or Cmd-E on Mac OS X) to enable, and Ctrl+D (or Cmd+D) to disable. Shortcut Ctrl+A (Cmd+A) selects all holes.

Field Feed specifies the feed for drilling operation (that is effectively a speed with which drill penetrates the laminate). Feed is specified in mm per minute.

For your convenience offsets that were entered on previous steps can be checked (and should it be necessary edited) in Offsets pop-out pane.

Misc pop-out pane contains several parameters that give you additional control over drilling process. It’s perfectly safe to leave them as is, if you aren’t sure.

Parameter Meaning
Clearance The height above the board, the program uses to move the tool between holes
Safety height The height above the board, where tool switches from rapid descend to feed

If you wish to have a look at G-Code that will be send to the machine, you can press a button with wrench icon in the toolbar.

When you are happy with all of the settings, press Go button. cirQWizard will send generated G-Code to the machine.

When this is don, press Continue button. If there are holes with different diameters on the board, cirQWizard will ask you to insert next drill and will present you a drilling screen for the next drill diameter.
