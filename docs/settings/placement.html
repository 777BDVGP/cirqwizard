Settings - Component Placement

Component placement settings include the following parameters:

Parameter Meaning Default value Units
Pickup height Z axis coordinate, the head descends to when picking up a component from panel mm
Move height Z axis coordinate, which is used when component is being moved above the board mm

All heights are specified in machine coordinate system.
