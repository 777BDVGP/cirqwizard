Latest version – 1.3.0

This is a new version with lots of performance improvements. Not only its faster than 1.1.xx versions, it is also capable of utilizing multi-core CPUs. So the better computer you have, the less you’ll have to wait for your tool paths to generate. A note for the existing users of the previous versions: most of the settings need to be reentered again (this is caused by a change in how they are stored internally).

Source code

The source code is available at git repository here: https://github.com/simonsalykov/cirqwizard

To get yourself the latest version of source code clone it with the following command:

git clone https://github.com/simonsalykov/cirqwizard.git

The easiest way to build sources is to use Apache Ant To compile the project run:

ant compile

and to run the application:

ant run
